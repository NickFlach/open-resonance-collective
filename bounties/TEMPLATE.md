## Submission Format

### Required Files
1. **Final Mix:** WAV or FLAC, minimum 44.1kHz/16-bit
2. **Stems:** Individual tracks as WAV/AIFF (recommended)
3. **Metadata:** JSON file with track info, creation notes, stem sources used
4. **Artist Statement:** Brief description of your creative choices and how they serve the bounty

### Naming Convention
```
bounty-[XXX]-[your-artist-name]-[track-title]
├── final-mix.wav
├── stems/
│   ├── kick.wav
│   ├── bass.wav
│   └── [etc...]
├── metadata.json
└── artist-statement.md
```

### Submission Process
1. Upload via [Submission Portal](../packages/submission-portal)

4. Post announcement in Discord #bounties channel
